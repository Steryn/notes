
### 2. 实践项目

```bash
# 启动所有高级示例
cd practise/advanced-examples
node start-all.js

# 单独启动性能优化示例
cd performance-optimization/memory-profiler
npm install
npm start

# 单独启动微服务示例
cd microservices/user-service
npm install
npm start

# 单独启动安全实践示例
cd security-practices/auth-system
npm install
npm start
```
